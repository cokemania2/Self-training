# Javascript 문법 정리

## method 

 ```String(int)``` >> int to String

 ```parseInt(String)``` >> String to int 

```String.split()``` >> String to array 

```String.indexOf()``` >> 문자열의 문자의 위치 반환 (문자가 없을시 -1 반환)

```String.lastIndexOf()```  >> 반대방향 부터 위치 반환

```Math.max(a,b)``` >> 최댓값 반환
